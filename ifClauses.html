Oraciones Condicionales (If Clauses)

Las oraciones condicionales se utilizan para expresar que la acción de una parte de la oración depende de la condición presentada en la otra parte. Existen varios tipos de oraciones condicionales en inglés, que varían según la probabilidad de que la condición se cumpla.

Estructuras de las Oraciones Condicionales

Tipo Estructura Ejemplo Traducción
Condición Cero (Real) If + presente simple, presente simple If you heat water, it boils. Si calientas agua, hierve.
Primera Condición (Posible) If + presente simple, futuro simple If it rains, I will stay at home. Si llueve, me quedaré en casa.
Segunda Condición (Hipotética) If + pasado simple, condicional simple If I had money, I would buy a car. Si tuviera dinero, compraría un coche.
Tercera Condición (Imposible) If + pasado perfecto, condicional perfecto If I had known, I would have called you. Si hubiera sabido, te habría llamado.

Ejemplos Detallados de Oraciones Condicionales

Tipo Ejemplo de Condición Ejemplo de Resultado Explicación Traducción
Condición Cero If you press the button, the light turns on. Se usa para expresar verdades generales. Si presionas el botón, la luz se enciende.
Primera Condición If it is sunny, we will go to the park. Se usa para hablar de una posibilidad futura. Si está soleado, iremos al parque.
Segunda Condición If I were you, I would take the job. Se usa para dar consejos o hablar de situaciones hipotéticas. Si fuera tú, tomaría el trabajo.
Tercera Condición If they had studied, they would have passed the exam. Se usa para hablar de situaciones hipotéticas en el pasado. Si hubieran estudiado, habrían pasado el examen.
